# Follow The Pi Self-Hosted Playbook

If the task involves any of the following, read `PI_SELF_HOSTED_AGENT_PLAYBOOK.md` first and follow its phase order:

- removing Firebase
- finishing the Prisma or NextAuth migration
- self-hosting this app on the Raspberry Pi
- adding machine APIs for list updates
- integrating NanoClaw

Do not restart from the old Firebase architecture, and do not discard unrelated local changes in the current worktree.
